# Auth Interceptor App

Angular-приложение, демонстрирующее работу HTTP-интерсептора для авторизации.

## Запуск

```bash
npx nx serve auth-interceptor-app
```

Открыть: http://localhost:4200

## Что внутри

- HTTP-интерсептор для добавления токенов авторизации к запросам
- Обработка 401/403 ответов от сервера
