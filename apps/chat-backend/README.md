# Chat Backend

NestJS-сервер чата с поддержкой WebSocket.

## Запуск

```bash
npx nx serve chat-backend
```

Сервер: http://localhost:3000

## Что внутри

- WebSocket-гейтвей для обмена сообщениями в реальном времени
- CORS включён для всех источников
